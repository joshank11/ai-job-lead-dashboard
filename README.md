# AI Job Lead Intelligence Dashboard

A Streamlit-based dashboard to safely discover recruiters, hiring managers, and alumni leads using Google Search instead of direct LinkedIn scraping.

## Features

- HR lead discovery
- Hiring manager discovery
- Alumni/network-based search
- AI lead scoring
- Excel export
- Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- Pandas
- Serper API

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
